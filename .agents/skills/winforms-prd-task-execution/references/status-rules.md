# Status Rules

## Movimentacao permitida

- Itens implementados saem de `Tarefas Pendentes:`.
- Itens implementados entram em `### Tarefas Finalizadas aguardando validacao Humana:`.
- Itens entram em `### Tarefas Concluidas e verificadas por Humano:` somente com pedido explicito.

## Regras praticas

- Mover apenas tarefas realmente implementadas.
- Nao antecipar status por expectativa de que "deve funcionar".
- Se a tarefa foi parcialmente entregue, mover apenas a parte concluida ou reescrever o detalhamento para refletir a realidade.
- O PRD deve representar o estado atual, nao o estado desejado.
